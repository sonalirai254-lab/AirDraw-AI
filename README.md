# 🎨 AirDraw AI

AirDraw AI is a Python-based AI project that lets users draw in the air using hand gestures. It uses OpenCV and MediaPipe for real-time hand tracking and gesture recognition.

---

## 🚀 Features

- ✋ Real-time Hand Tracking
- 🎨 Air Drawing
- 🌈 Multiple Color Selection
- 🖌️ Brush Size Control
- 🧽 Eraser Tool
- ↩️ Undo / Redo
- 💾 Save Drawing
- 🗑️ Clear Canvas
- 🔺 AI Shape Detection
- 📷 Webcam Support
- 🖥️ User-Friendly Toolbar

---

## 🛠️ Technologies Used

- Python 3
- OpenCV
- MediaPipe
- NumPy

## ⚙️ Installation

```bash
git clone <repository-url>
cd AirDraw-AI
pip install -r requirements.txt
python main.py
```

---

## 🎮 Controls

- Draw Gesture → Draw
- Select Gesture → Select Tool
- Clear Gesture → Clear Canvas
- S → Save Drawing
- U → Undo
- R → Redo
- D → Detect Shape
- C → Clear Canvas
- Q → Quit
